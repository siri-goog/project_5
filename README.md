## Setup
To run this project, install it locally using npm:
```
$ npm init -y
$ npm install -s express
$ npm install -s ejs
$ npm install -s morgan
$ npm install -s body-parser
$ npm install -s cookie-parse
$ npm install -s express-session
$ node app.js

```
